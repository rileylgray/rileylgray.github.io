My Work

Below are some projects I have worked on. These include schools projects as well as my own personal projects I have worked on. Some can even be found on the Play Store! Click the first icon for any game to view it.

Dodge Square

A simple game made in Java, with the goal of dodging and staying alive as long as possible. I keep adding to the game so stay with me. The game includes major and minor enemies. Some chase you, some shoot at you, some just bounce around hoping to hit you. You can upgrade yourself but each upgrade reduces your score so play smart and score big. Hard mode is currently in development.Click on the Github icon to view my code

Sheridan Book Store

A small website I made for a school project. The website allows for the user to add, edit and remove books to a bookstore. Using a database the website is stored and is apply to be pulled from. This was made with the help of Apache Tomcat. Click on the Github icon to view my code

Swim Fish Swim

This is my first real attempt at creating a game for use on an android device. I created the pixel art and coded the logic for this game and it is fully published on the Play Store. I am proud to say that my game was trending on the number 9 slot on the play store. Click the android icon to be taken to my app

Bounce Ball Bounce

This is my first game using unity. It is also my first real experience coding in C#. Since C# is similar to Java I was able to pick this up fairly quickly. I hope to expand upon this game in the future and make it more than a pong like game. Click the android icon to be taken to my app

About Me

My Name is Riley Gray. I am a Sheridan College student in the Software Development and Network Engineering program.

After graduating high school, I headed straight to Humber College and enrolled in the Electrical Techniques program. This was a one-year program in which I graduated with honours. After this program and a few months of work I decide to head to Sheridan College to pursue software development. I am currently in my 3rd year of my 3-year program and have already learned languages like Java, HTML, CSS, Javascript, and MySQL. I am learning even more through Sheridan and my own pursuits.

My work experience now includes two 4 month co-op terms. The first at the county of wellington and the second at the Region of Halton. Both I was given excellent reviews by my supervisors. During these terms I was working as a IT support technician. This was excellent experience but I hope to find a developer job in the future.
